# あんもくのルート
分かりにくい道を分かる人が解説（投稿）するアプリです
**地元の人にしか分からない道が多い**という印象から**「暗黙のルール」**にかけたタイトルになっています
[![Image from Gyazo](https://i.gyazo.com/9493f066690f803f6f243aaf20aae528.gif)](https://gyazo.com/9493f066690f803f6f243aaf20aae528)
